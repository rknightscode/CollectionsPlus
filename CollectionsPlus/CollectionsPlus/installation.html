Installation guide

🌐 Microsoft Edge (any recent version)
🪟 Windows 10 or 11
⏱ About 10 minutes
Part 1

Download the extension files

1
Download the latest release
Go to the releases page on GitHub and download CollectionsPlus.zip from the Assets section.
Do not use the green "Code → Download ZIP" button on the main repo page — that downloads the source, not the packaged release.
2
Extract the ZIP
Right-click the downloaded ZIP and choose Extract All. Choose somewhere easy to find, such as your Documents folder. You will end up with a folder called CollectionsPlus — remember where it is.
Part 2

Download the sql.js library files

3
Go to the sql.js releases page
Open github.com/sql-js/sql.js/releases/latest in your browser. Scroll past the release notes to the Assets section and download sqljs.zip.
Do not click "Source code (zip)" — that is not the right file.
4
Copy two files into your CollectionsPlus folder
Open the downloaded sqljs.zip and copy these two files into your CollectionsPlus folder:
CollectionsPlus/
  manifest.json
  popup.html
  popup.js
  popup.css
  db.js
  background.js
  sql-wasm.js    ← copy this in
  sql-wasm.wasm  ← and this
  icons/
Part 3

Load the extension into Edge

5
Open Edge extensions
In the Edge address bar, type edge://extensions and press Enter.
6
Turn on Developer mode
In the top-right corner of the extensions page, toggle Developer mode on. Two new buttons will appear.
7
Load the extension folder
Click Load unpacked, navigate to your CollectionsPlus folder, and click Select Folder. Collections+ will appear in the extensions list.
8
Pin Collections+ to the toolbar
If you don't see the C+ icon in the Edge toolbar, click the puzzle piece (🧩) icon and pin Collections+ from the list.
Done. The extension is installed. Continue to first launch setup below.
Part 4

First launch setup

9
Open Collections+
Click the C+ icon in the toolbar. The sidebar will open and show the setup screen.
10
Choose a storage folder
Click Choose Folder and select a folder on your computer where you want Collections+ to store its database. A file called collections.db will be created there.
A good choice is your Documents folder or a subfolder inside it. Avoid choosing a folder inside your OneDrive — that folder is used separately for backups.
11
You're ready
Collections+ will open and show your (empty) collections list. Create your first collection with + New and start saving pages.
Tip: Each time you start Edge, click Unlock Collections when you first open the sidebar. This is a one-click browser security requirement — Edge asks you to confirm folder access once per session.
Troubleshooting

Common issues

I don't see the C+ icon in the toolbar
Click the puzzle piece (🧩) icon in the Edge toolbar, find Collections+ in the list, and click the pin icon next to it.
"Unlock Collections" appears every time I start Edge
This is normal and by design. Edge requires you to confirm folder access once per browser session. Click the button — it takes one second and your collections load immediately.
A saved page shows a black or blank screenshot
Some pages block screenshots for security reasons (banking sites, Edge's own settings pages). The page is saved correctly with its title and address — only the thumbnail is missing. You can leave it blank or refresh it when you're on a page where capture is allowed.
The extension shows an error after I update the files
Go to edge://extensions, find Collections+, and click the reload (↺) button. This refreshes the extension with the new files.
Edge asks to confirm new permissions after an update
This is expected when a new version adds permissions (such as right-click context menu or history access). Review the permissions prompt and click Accept to continue using the extension.
I accidentally deleted a collection
If you have a recent backup in OneDrive, restore from it via the Settings panel (⚙ icon). This is a good reason to back up regularly before making bulk changes.
